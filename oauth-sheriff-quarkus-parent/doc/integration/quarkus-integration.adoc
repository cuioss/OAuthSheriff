== Overview

The oauth-sheriff-quarkus extension integrates the JWT validation library with Quarkus, providing automatic configuration, health checks, metrics, and DevUI integration.

== Architecture

The extension follows standard Quarkus extension architecture with separate deployment and runtime modules:

* **Deployment Module**: Build-time processing, reflection registration, DevUI setup
* **Runtime Module**: Configuration classes, health checks, metrics integration

=== OAuthSheriffProcessor

Handles build-time processing:

* Registers the `oauth-sheriff` feature
* Sets up reflection for `JwtValidationConfig`, `TokenValidator`, `IssuerConfig`, etc.
* Registers `HttpJwksLoader` for runtime initialization (native image support)
* Configures DevUI components for development mode

=== OAuthSheriffRecorder

Leverages CDI for component management. All runtime initialization is handled by CDI; health checks are automatically discovered through annotations (`@ApplicationScoped`, `@Readiness`, `@Liveness`).

=== Health Checks

* `JwksEndpointHealthCheck` (`@Readiness`): Monitors JWKS endpoint connectivity
* `TokenValidatorHealthCheck` (`@Liveness`): Validates TokenValidator configuration

Automatically discovered by Quarkus -- no explicit registration needed. See xref:health-checks.adoc[Health Checks] for details.

=== Metrics Integration

Automatically exposes JWT validation error counters, security event monitoring, and performance metrics when Micrometer is present. See xref:metrics-integration.adoc[Metrics Integration] for details.

== Configuration

Uses MicroProfile Config API with the `sheriff.oauth` prefix:

[source,properties]
----
sheriff.oauth.enabled=true
sheriff.oauth.parser.allow-unsafe-algorithms=false
sheriff.oauth.parser.max-token-size-bytes=8192
sheriff.oauth.issuers.keycloak.issuer-identifier=https://keycloak.example.com/auth/realms/master
sheriff.oauth.issuers.keycloak.jwks.http.url=https://keycloak.example.com/auth/realms/master/protocol/openid-connect/certs
----

== DevUI Integration

In development mode (`@BuildStep(onlyIf = IsDevelopment.class)`):

* `OAuthSheriffProcessor.createJwtDevUICard()`: Creates DevUI card pages
* `OAuthSheriffProcessor.createJwtDevUIJsonRPCService()`: Registers JSON-RPC service
* `OAuthSheriffDevUIRuntimeService`: Provides runtime data access
